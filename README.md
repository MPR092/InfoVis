# Information Visualization - Earthquake Data Visualization
----------------------------------------------------------------

## Download

Download the project folder in a seperate directory.

Open the directory in vscode/editor.

Open terminal and switch to the project directory.

----------------------------------------------------------------

## Now, First install all the required associated dependencies by using following cmd.


### `npm install`

----------------------------------------------------------------

## Then run the react app using


### `npm start`


## The system should run on the localhost in the browser.
